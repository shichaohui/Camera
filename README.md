# camera

基于 Camera 和 Camera2 实现的相机组件。

## 特性

* 切换摄像头。
* 切换闪光灯状态。
* 拍照(.jpg)。
* 拍视频(.mp4)。
* 点击对焦。
* 双指缩放预览内容。
* 基于 Camera 和 Camera2 实现。

## 使用默认 UI

若对 UI 没有特殊要求，可直接 [使用 camera-ui](/camera-ui/) 一键接入相机组件。

## 自定义 UI

[camera API 库](/camera/) 提供上面提到的所有功能相关 API，配合自定义 UI 直接调用即可。
